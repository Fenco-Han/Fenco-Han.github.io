时间复杂度和空间复杂度

Big O notation

  • O（1）:常数复杂度
  • O（log n）:对数复杂度
  • O（n）：线性时间复杂度
  • O（n^2）:N 平方
  • O（n^3）:N 立方
  • O（2^n）:指数O（n!）:阶乘

注意：只看最高复杂度的运算。

O(1):
    int n = 100;
    System.out.println("hello " + n);

O(1):    
    int n = 1000;
    System.out.println("hello " + n);
    System.out.println("hi " + n);
    System.out.println("helo " + n);

O(N):
    for(int i = 1; i<= n; i++) {
        System.out.println("hello " + i);
    }

O(N^2):
    for (int i = 1; i<=n; i++) {
        for (int j = 1; j<=n; j++) {
            System.out.println("hello " + i + j);
        }
    }

O(log(n)):
    for (int i = 1; i < n; i = i * 2) {
        System.out.println("hello " + i);
    }

O(k^n):
    int fib(int n) {
        if(n <=2 ) return n;
        return fib(n - 1) + fib(n - 2); 
    }
